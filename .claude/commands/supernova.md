# /supernova — Morning Briefing

## What It Does
Starts the session. Reads primer.md, surfaces priorities, sets the day's agenda.

## Steps
1. Read `primer.md`
2. List: done recently / next steps / blockers / finance status
3. Suggest today's focus (top 1-3 actions)
4. Ask: "What are we working on today?"

## Output
```
SUPERNOVA — [date]
━━━━━━━━━━━━━━━━━
DONE:     [from primer]
NEXT:     [from primer]
BLOCKERS: [from primer]
FINANCES: [from primer]
━━━━━━━━━━━━━━━━━
TODAY'S FOCUS: [top suggestion]
What are we working on?
```
