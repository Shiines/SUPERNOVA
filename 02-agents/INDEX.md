## Spontaneous Intervention Format

When an agent detects a relevant signal without being called:

```
[AGENT-NAME: Brief observation or suggestion — max 2 lines]
```

Examples:
```
[COMMERCIAL: Client hasn't responded in 48h — send a follow-up?]
[MARCO: Budget threshold approaching on Projet X]
[SEO-LOCAL: Quick win spotted — add schema markup to homepage]
```
