# LIOR: Dubai Agency Targeting Methodology

> Objectif : 100 agences qualifiées pour les DMs du lundi
> Priorité : Owners et brokers indépendants. Pas agents salariés.

---

## FILTRE DE QUALIFICATION

### Critères INCLUDE

| Critère | Pourquoi |
|---------|----------|
| Owner de l'agence ou broker indépendant | Peut signer sans approbation. Décide seul. |
| Quartiers premium Dubai | Palm, Marina, Downtown, DIFC, JBR, Hills, Jumeirah Bay |
| Listings sans vidéo sur Property Finder / Bayut | Signal direct : besoin non adressé |
| Compte Instagram actif (>500 followers, posts récents) | Signifie qu'il investit dans sa présence |
| 5 à 50 agents dans l'agence | Assez grand pour avoir du volume, assez petit pour décider vite |

### Critères EXCLUDE

| Critère | Pourquoi |
|---------|----------|
| Agent salarié d'une agence internationale (Savills, JLL, CBRE, Knight Frank) | Cycle de décision interne trop long |
| Agence off-plan uniquement (Emaar, Damac agents) | Budget marketing géré par le developer |
| Listings exclusivement <500k AED | Pas le profil premium LIOR |
| Compte Instagram inactif ou personnel | DM froid = bruit |

---

## MÉTHODE DE SOURCING

### Étape 1: Property Finder / Bayut (filtre sans vidéo)

1. Aller sur propertyfinder.ae ou bayut.com
2. Filtre : Residential Sale + Premium neighborhoods
3. Trier par "Recently listed"
4. Identifier listings sans video tour
5. Cliquer sur l'agence → noter nom + contact

**Signe prioritaire :** propriétés délistées depuis >30j = douleur de vente visible

### Étape 2: Instagram (enrichissement)

1. Chercher hashtags : #dubaiproperties #dubaimarina #palmjumeirah #dubairealestate
2. Filtrer les comptes avec :
   - Bio = "real estate" + numéro WhatsApp visible
   - Posts = photos propriétés (pas lifestyle)
   - Engagement réel (pas 50K followers + 3 likes)
3. Vérifier : est-ce un owner ou un agent salarié ?

**Signal owner :** compte personnel qui représente aussi l'agence / bio dit "Founder" / "MD" / "Principal"

### Étape 3: LinkedIn (vérification ownership)

1. Chercher le nom de l'agence
2. Vérifier : la personne est-elle "Founder", "CEO", "Managing Director", "Owner" ?
3. Pas "Associate", "Senior Agent", "Consultant"

## PRIORISATION

### Tier 1: Contacter lundi (5 contacts)
- Amies/contacts directs du CEO dans l'immobilier Dubai
- Probabilité de réponse : haute
- Script : DM semi-warm (mentionner la connexion personnelle)

### Tier 2: DMs lundi-mardi (15-20 contacts)
- Owners identifiés via Property Finder (sans vidéo, listing récent)
- Instagrameurs actifs dans les quartiers premium

## TIMING DM

**Meilleurs créneaux Dubai (GMT+4) :**
- Matin : 9h–11h (avant réunions client)
- Soir : 18h–20h (après les visites)
- Éviter : vendredi après-midi, samedi matin (repos/famille)

**Volume recommandé :**
- Lundi : 5 DMs chauds (amis CEO) + 15 DMs froids
- Mardi-jeudi : 10–15 DMs/jour
- Pas plus de 30 DMs/jour par compte Instagram (risque shadow ban)
