# LIOR — Scripts Outreach
## Cold DM · Talk Track · WhatsApp Response Flow

> Zéro preuve sociale fictive. Zéro prix visible. Zéro "drone" / "FPV" / "AI".
> Strategie : warmth + specificity + visual proof (dès que demo tour disponible)

---

## SCRIPT 1 — Cold DM Instagram (Dubai agents)

**Cible :** Owners/brokers, propriétés premium sans vidéo. Pas les agents salariés.

**SANS demo tour disponible :**

```
Hi [Prénom],

Noticed your listing at [adresse/quartier].
Most agents here are still presenting empty rooms.

We produce cinematic property tours for agents in [quartier].
24-72h from your photos. Nothing on-site.

Can I send you an example?

[Prénom] · LIOR
```

**AVEC demo tour disponible :**

```
Hi [Prénom],

I made a 30-second version for a similar unit in [quartier comparable].
Want me to drop the link here?

[Prénom] · LIOR
```

**Pourquoi ce format (AVEC demo) :**
- "I made" → passé accompli. Pas "je peux faire" — c'est fait.
- "30-second version" → engagement cognitif nul. 30 secondes = pas de friction.
- "for a similar unit" → signal de pertinence directe à leur situation.
- "Want me to drop the link here?" → question fermée oui/non. Dire oui ne coûte rien.
- Pas de lien dans le premier message → évite le filtre spam, crée la curiosité.
- Quand ils disent oui → micro-engagement = taux de visionnage supérieur.

**Règles DM :**
- Une seule phrase d'accroche (property spécifique, pas générique)
- Jamais plus de 5 lignes visibles sans défilement
- Pas de liste de services
- Pas de prix
- **Finir par une question FERMÉE (oui/non) — jamais une question ouverte**
- "Interested?" est interdit — force une décision trop tôt
- "Worth a quick look?" est interdit — trop doux pour le marché Dubai

```
so you can judge the quality directly.

Just let me know.

[Prénom] · LIOR
```

**Règles :** Un seul follow-up. Pas de troisième relance froide.

## SCRIPT 3 — Talk Track Visite Samui (SUPERNOVA — pas LIOR)

**Contexte :** Agences immobilières Koh Samui / Thaïlande. Pitch web agency (SUPERNOVA).
Ne pas mentionner LIOR. Ne pas mentionner Dubai.

**Ouverture :**
```
Hi, I run a web agency specializing in real estate.
I was looking at properties in [quartier] and noticed
most agencies here don't have a strong online presence.

We build sites that actually bring in leads — fast load,
mobile-first, WhatsApp-ready. Most agencies see results
within the first month.

Can I show you what we typically deliver?
```

**Si intérêt :**
```
We do a full site for [prix SUPERNOVA à définir par CEO].
Delivered in [délai SUPERNOVA].
Everything handled — design, copy, domain, hosting.

I can have a proposal ready for you by [date J+2].
```

**Règles :**
- Pitch court (90 secondes max)
- Avoir le portfolio sur téléphone (sites existants SUPERNOVA)
- Objectif visite = rendez-vous de suivi, pas clôture
- Note : ne pas donner de devis sur place sans avoir qualifié le budget

## SCRIPT 4 — WhatsApp Response Flow (3 messages)

**Déclencheur :** Agent répond au DM Instagram ou contacte via site LIOR.

### Message 1 — Réponse immédiate (sous 2h)

```
Hi [Prénom],

Thanks for reaching out.

Quick question to make sure we can help:
Do you have a listing with empty rooms you'd like
to present at their full potential?

[Prénom] · LIOR
```

### Message 2 — Si oui (qualification)

```
Got it.

To get started, I just need:
→ A few photos of the rooms (one per room)
→ Neighborhood and property type (apartment / villa / studio)
→ Target buyer profile (local / international / investor)

That's it. We handle everything from there.
Delivery in 72h from the brief.
```

### Message 3 — Confirmation reception brief

```
Perfect — received everything.

We'll send you the first direction within [24/48]h.
Any questions in the meantime, I'm here.

LIOR Visual Studio
```

**Règles WhatsApp :**
- Zéro "drone" / "FPV" / "AI" / "luxury" / tiret cadratin
- Jamais envoyer les 3 messages d'un coup
- Toujours attendre la réponse avant le message suivant
- Ton professionnel + warm (pas corporate, pas commercial)

---

## SIGNAL DE QUALIFICATION

**Répondre en priorité si le prospect :**
- Mentionne un délai ("launch next week", "urgent")
- A plusieurs propriétés ("portfolio", "multiple listings")
- Est owner/broker (pas salarié)
- Localisation premium (Marina, Palm, Downtown, DIFC)

**Ne pas poursuivre si :**
- Agent salarié d'une grande agence internationale (cycle trop long)
- Budget clairement hors cible (cherche "cheap" ou "affordable")
- Pas de propriétés disponibles maintenant
